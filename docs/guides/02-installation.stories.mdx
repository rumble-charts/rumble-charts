# Installation

Install via NPM

```
npm install --save rumble-charts
```

or Yarn

```bash
yarn add rumble-charts
```

Import

```typescript
import {
  // main component
  Chart,
  // graphs
  Bars, Cloud, Dots, Labels, Lines, Pies, RadialLines, Ticks, Title,
  // wrappers
  Layer, Animate, Transform, Handlers,
  // helpers
  DropShadow, Gradient
} from 'rumble-charts';
```
